## About The Project

---

###### _._

  <div align="center">

  <a href="https://github.com/JPLACLAU/Hunger-Games">  
    <img src="images/400x400.jpg" alt="Wake UP Contract"    width="180" height="180">

  </a>  
    
  </div>

###### _._

This is a project regarding a SmartContract to use among friends when going to a bar to leave a collaborative tip to the staff of the venue.
This project is in an early stage.
Be very careful if you use this code for any purposes, I advise against it.

Here's why a group of waiters and bartenders should use it in their bar:

- Fair tipping is complicated
- Not everyone really shares
- As it is open in a public ledger, there is no room for mistrusting amongs collegues.

This project is in an early stage.
Be very careful if you use this code for any purposes, I advise against it.
